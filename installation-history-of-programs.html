Pelican

  • why a static site generator,written in python
  • Installation: pip install pelican
  • date:2013-11-08

mczip

  • why 带中文字符文件名的zip压缩包打开用unzip解压后，文件名和目录名都是乱码，gooogle一下，很多给出的是用unzip -O CP639这样的命令，但是现在unzip 好像没有了-O这个参数。还有很多是推荐用convmv 转换编码格式，不知道为什么不管用。然后在这个博客找到了这个小软件mczip
  • Installation: /usr/local/bin/mczip
  • date: 2013-11-30

texlive

  • why 起因是翻译矩阵论文，里面有很多公式，在linux环境下怎么高效得去编辑含有公式的文档呢，于是想到了tex，做毕业设计的时候大东就是用它写的论文。
  • Installation: /usr/home/smallest/texlive/2013
  • date: 2013-12-05

jhead

  • why 需要旋转图像的方位来保证其在浏览器中的正确显示.jhead可以修改文件的Exif(Exchangeable Image File,is a kind of JPEG)头部信息。
  • Installation:apt-get install jhead
  • date: 2013-12-14

libpdf(chrome)

  • why 用chromium看pdf文档，但是chromium不带这项功能，从chrome的安装包中提取libpdf放到/usr/lib/chromium
  • Installation:ar x google-chrome-stable;tar --lzma -xvf data.tar.lzma;cp opt/google/chrome/libpdf.so /usr/lib/chromium;
  • date: 2013-12-14

gdb

  • why 想着用gdb调试一下程序，结果居然没有这个程序，让我很惊讶，这不是linux系统的标配吗，想想自己老说喜欢编程，居然这么就半年多都没调过C和C++的程序，基本没编多少行代码，得找找开源的项目，多做贡献。
  • apt-get install gdb
  • date: 2013-12-19

fcitx-module-cloudpinyin

  • why fcitx的一个插件，提供来自云(google,QQ,sougou,baidu)的候选词，提供更加顺畅的输入体验
  • apt-get install fcitx-module-cloudpinyin
  • date: 2014-02-05

flash 插件

  • why 浏览器需要flash插件
  • 在http://get.adobe.com/flashplayer/?no_redirect 网站下载Adobe Flash 的源包install_flash_player_11_linux.i386.tar.gz。 tar -zxvf ....解压文件。将libflashplayer.so文件拷贝到了/usr/lib/chromium/plugins/文件夹下，将解压出来的usr文件夹下的东西拷贝到系统的/usr文件夹下cp -r usr/* /usr 创建文件夹~/.mozilloa/plugins.将libflashplayer.so拷贝到该文件夹.两个浏览器同时开时,只有一个能听豆瓣fm.
  • date:2013-6-21
